\begin{abstract}{Developement for mobile platforms using backend as a service: 
    case study of the Minha UFG application}
    
The development of mobile and web applications has been facilitated by companies that provide cloud backend services. Here are presented the core challenges and their solutions, arising from the use of those services. The time gained in the development process of these applications compensates the use of these services. 
\end{abstract}
